# Plan: Migrate from Husky to Lefthook

## Overview

Replace Husky (Node) + commitlint (Node) with Lefthook (Go) + conventionalcommit/commitlint (Go). Fully removes Node.js from the project's dev toolchain.

## Tool Installation

Add a `tools` Makefile target that installs both tools:

```makefile
.PHONY: tools
tools:
	go install github.com/evilmartians/lefthook@latest
	go install github.com/conventionalcommit/commitlint@latest
	go install github.com/vektra/mockery/v3@latest
```

## Files to Create

### `lefthook.yml`

```yaml
pre-commit:
  parallel: true
  commands:
    lint:
      glob: "*.go"
      run: make lint
    notice:
      glob: "go.{mod,sum}"
      run: go-licenses report . --template=notice.tpl --include_tests > NOTICE && git add NOTICE
    docs:
      glob: "cmd/**/*.go"
      run: go run internal/tools/docgen/main.go --out ./docs/cli --format markdown && git add docs/cli/

commit-msg:
  commands:
    commitlint:
      run: cat {1} | commitlint lint

pre-push:
  commands:
    test:
      run: make test
```

### `.commitlint.yaml`

Generated by `commitlint config create --all`. Covers the same rules as `@commitlint/config-conventional` (type-enum, min/max lengths, etc.). Committed as-is; can be tweaked later.

## Files to Delete (tracked by git)

| File | Reason |
|---|---|
| `.husky/commit-msg` | Logic moved to lefthook.yml |
| `.husky/pre-commit` | Logic moved to lefthook.yml |
| `.husky/pre-push` | Logic moved to lefthook.yml |
| `.husky/post-commit` | Commented-out code, no-op |
| `package.json` | No more Node devDependencies |
| `pnpm-lock.yaml` | No more Node devDependencies |
| `.commitlintrc.json` | Replaced by `.commitlint.yaml` |

## `.gitignore` Changes

| Line | Action |
|---|---|
| `.husky/_/` | Remove — entire `.husky/` will be deleted |

## Implementation Steps

1. Add `tools` target to Makefile
2. Run `make tools` to install lefthook and commitlint
3. Create `lefthook.yml`
4. Generate `.commitlint.yaml` via `commitlint config create --all`
5. Delete tracked files: `git rm .husky/commit-msg .husky/pre-commit .husky/pre-push .husky/post-commit package.json pnpm-lock.yaml .commitlintrc.json`
6. Edit `.gitignore`: remove `.husky/_/` line (entire `.husky/` directory gone)
7. Run `lefthook install` to register git hooks
8. Verify: stage a change, commit, push — hooks fire correctly
